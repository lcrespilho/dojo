## Config file
Create at home ```.dojo_4.json``` using this config:

```
{
    "LOG_PATH": "<PATH>/log",
    "DB_NAME": "dojo_4_db",
    "DB_USER": "<DB_USER>",
    "DB_PASSWORD": "<DB_PASSWORD>"
}
```
